Standalone Grid World Experiment

About this experiment:

This is a standalone version of the grid world experiment that runs without nodeGame dependencies. It includes all the same functionality as the original experiment but runs independently in the browser.

Required Dependencies:

Make sure the following files are loaded before this script:

  • utils.js - Game utilities and constants
  • rlAgent.js - AI agent implementation
  • nodeGameHelpers.js - Canvas drawing functions
  • Map data files (MapsFor1P1G.js, MapsFor1P2G.js, etc.)
