Human-Human Collaboration Experiments

Welcome to the real-time multiplayer collaboration experiments!

Available Experiments:

2P2G: Two players, two goals - players must coordinate to reach goals

2P3G: Two players, three goals (third goal appears during game)

Quick Start:

Start Test Interface Dependency Test

How to Test:

  1. Click "Start Test Interface" to open the test page
  2. Open the test page in two different browser windows/tabs
  3. Select the same game type (2P2G or 2P3G) on both windows
  4. Choose movement mode (simultaneous or turn-based)
  5. Click "Join Game" on both windows
  6. Wait for both players to connect
  7. Click "Ready" on both windows to start the game
  8. Use arrow keys or WASD to move and collaborate!

Features:

  • Real-time Multiplayer: Two human players can play together
  • Movement Modes: Simultaneous or turn-based movement
  • Multiple Trials: Full experiment with multiple trials
  • Progress Tracking: Visual progress bar and trial counter
  • Collaboration Scoring: Success when both players reach the same goal
  • Dynamic Goals: 2P3G experiments feature a third goal that appears during gameplay
